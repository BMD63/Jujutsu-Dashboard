# Jujutsu Dashboard

Дашборд для мониторинга аномалий (духов) в реальном времени.

## Текущий прогресс

✅ **Базовая настройка завершена:**
- Next.js 14 с TypeScript
- Docker контейнеризация (dev среда)
- Базовая структура Feature Sliced Design
- Страница мониторинга (/monitoring)

## Запуск

```bash
# Запуск в development режиме
docker-compose up

# Или с пересборкой
docker-compose up --build
